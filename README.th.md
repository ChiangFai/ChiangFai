# การวิเคราะห์เชิงพื้นที่แบบเปิดของรูปแบบการเผาไหม้ทางการเกษตรในภาคเหนือของประเทศไทย

**2018–2026 · จังหวัดเชียงใหม่ · นิติวิทยาศาสตร์การเผาไหม้จากดาวเทียม**

โครงการนี้ใช้ข้อมูลดาวเทียมที่เปิดให้ใช้งานฟรี (NASA FIRMS, Copernicus Sentinel-2) และ Google Earth Engine เพื่อทำแผนที่ วัดปริมาณ และวิเคราะห์รูปแบบไฟป่าและการเผาพื้นที่ทางการเกษตรทั่วจังหวัดเชียงใหม่ในช่วงระยะเวลาแปดปี

เป้าหมายคือการผลิตชุดข้อมูลเชิงพื้นที่ที่ทำซ้ำได้และตรวจสอบได้สาธารณะ ซึ่งนักวิจัย นักข่าว NGO หรือหน่วยงานภาครัฐใดก็ตามสามารถตรวจสอบได้อย่างอิสระ

---

## สิ่งที่โครงการนี้ทำ

| โมดูล | วัตถุประสงค์ |
|---|---|
| `retrospective_analysis.py` | การทำแผนที่พื้นที่เผาไหม้รายปีผ่าน Sentinel-2 dNBR (2018–2026); ส่งออก raster การเกิดซ้ำของการเผาไหม้ |
| `real_time_alerts.py` | ดึงข้อมูล NASA FIRMS API สำหรับการตรวจจับไฟที่กำลังลุกไหม้; บันทึก CSV snapshots พร้อมการประทับเวลา |
| `clustering.py` | DBSCAN spatial clustering บนจุดไฟ; ตรวจสอบเหตุการณ์กลุ่มที่เกิดขึ้นพร้อมกันตามเวลา |
| `dashboard/app.py` | Streamlit dashboard สำหรับการสำรวจเชิงโต้ตอบ |

## เริ่มต้นใช้งาน

```bash
pip install -r requirements.txt
```

**ยืนยันตัวตน Google Earth Engine (ครั้งเดียว):**
```python
# ใน gee_setup.py ให้ยกเลิกการคอมเมนต์ ee.Authenticate() แล้วรันครั้งเดียว
python gee_setup.py
```

**รับ NASA FIRMS API key ฟรี:**
→ https://firms.modaps.eosdis.nasa.gov/api/map_key/
ตั้งค่าใน `real_time_alerts.py` เป็น `MAP_KEY`

**รันการวิเคราะห์ย้อนหลัง:**
```bash
python retrospective_analysis.py
```
คำสั่งนี้จะจัดคิว GEE export ไปยัง Google Drive ของคุณ (scale 10m, ใช้เวลาประมาณสองสามชั่วโมง)

**ดึงข้อมูลไฟล่าสุด:**
```bash
python real_time_alerts.py
```

**จัดกลุ่มและตรวจสอบเหตุการณ์ที่มีการประสานงาน:**
```bash
python clustering.py
```

**เปิด dashboard:**
```bash
streamlit run dashboard/app.py
```

## หมายเหตุเกี่ยวกับวิธีการ

- **ค่าเกณฑ์ dNBR:** ใช้ `> 0.2` สำหรับการจำแนกพื้นที่เผาไหม้ ไฟเพื่อการเกษตรในเขตร้อนอาจต้องใช้ค่าเกณฑ์ที่ต่ำกว่า (0.1–0.15); ปรับใน `get_burned_area()`
- **ฤดูกาลเผาไหม้:** ภาพก่อนไฟเดือนพฤศจิกายน–ธันวาคม; ภาพหลังไฟเดือนมีนาคม–เมษายน ปรับสำหรับภูมิภาคอื่น
- **ตัวกรองความเชื่อมั่น FIRMS:** ไม่ได้ใช้งานโดยค่าเริ่มต้น — กรองที่ `confidence >= 'nominal'` สำหรับการวิเคราะห์ที่เข้มงวดขึ้น
- **Clustering:** DBSCAN ด้วย haversine distance ค่าเริ่มต้น `eps=5km`, `min_samples=3` ปรับแต่งสำหรับความหนาแน่นของไฟในพื้นที่
